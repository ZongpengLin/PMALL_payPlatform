# PMALL_payPlatform
PMALL系统的支付通用平台开发
> 以微信扫码、支付宝扫码为主要支付方式，开发者文档中： 微信 Native支付方式 和 支付宝PC移动端支付方式

## 支付开发者文档
详见官方：
- 微信： https://pay.weixin.qq.com/wiki/doc/apiv3/apis/chapter3_4_1.shtml
- 支付宝： https://opendocs.alipay.com/
